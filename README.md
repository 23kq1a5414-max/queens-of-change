## ✨ Extra Features Added

- 🎨 **Custom colour palette** — deep plum, warm gold & rose
- 📱 **Fully responsive** — mobile, tablet & desktop
- 🎞️ **Scroll reveal animations** — sections fade in as you scroll
- 🔢 **Animated stat counters** — numbers count up on scroll
- 🌐 **Fixed navigation bar** with smooth scroll
- 🃏 **Pillar cards** with hover accent effects
- 💬 **Testimonial section** with community voices
- ⚡ **Reduced motion support** for accessibility
- 👑 **Crown motif divider** as a signature design element
